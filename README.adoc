= OpenSCD Plugin: Cleanup
:plugin-name-kebab-case: oscd-editor-cleanup
:nofooter:
:uri-docs: https://openpowershift.github.io/oscd-editor-cleanup/docs
:uri-demo: https://openpowershift.github.io/oscd-editor-cleanup
:uri-sysconex: https://sysconex.openpowershift.com/
:uri-issues: https://github.com/openpowershift/oscd-editor-cleanup/issues

This plugin identifies and removes unreferenced SCL elements — DataSets, orphaned control blocks, and unused DataTypeTemplates entries — that accumulate over the lifecycle of an IEC 61850 configuration project.
