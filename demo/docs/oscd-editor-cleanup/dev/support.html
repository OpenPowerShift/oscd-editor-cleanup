Support

Reporting issues

Please report bugs, unexpected behaviour, and feature requests on the GitHub issue tracker.

When reporting a bug, include:

  • The version of the plugin (or the commit hash if using the CI build)

  • A minimal SCL file that reproduces the problem, if applicable

  • The browser and operating system you are using

  • A description of what you expected to happen and what actually happened

Community

Join the OpenSCD Zulip chat to discuss this plugin and other OpenSCD tools with the community.

Source code

The full source code is available at https://github.com/openpowershift/oscd-editor-cleanup.

Pull requests are welcome. See Development for instructions on how to build and test the plugin locally.
