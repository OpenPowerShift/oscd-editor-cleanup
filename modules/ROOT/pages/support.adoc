== Reporting issues

Please report bugs, unexpected behaviour, and feature requests on the {url-issues}[GitHub issue tracker].

When reporting a bug, include:

* The version of the plugin (or the commit hash if using the CI build)
* A minimal SCL file that reproduces the problem, if applicable
* The browser and operating system you are using
* A description of what you expected to happen and what actually happened
